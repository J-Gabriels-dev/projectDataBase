# projectDataBase

# 📚 Sistema de Gerenciamento de Biblioteca (MySQL)

Projeto final desenvolvido para a disciplina de Banco de Dados no **IFPI - Campus Angical**. O objetivo foi aplicar conhecimentos de modelagem relacional, criação de tabelas e manipulação de dados.

## 🛠️ Tecnologias Utilizadas
- **Banco de Dados:** MySQL
- **Modelagem:** draw.io
- **Linguagem:** SQL

## 📊 Estrutura do Projeto
O banco de dados contempla o gerenciamento de:
- **Alunos** (Cadastro e dados pessoais)
- **Livros e Autores** (Gestão de acervo)
- **Empréstimos** (Registro de saídas e devoluções)
- **Multas** (Regras de negócio para atrasos)

## 🚀 Como executar
Os scripts foram divididos para facilitar a manutenção:
1. `Criacao.tabelas.sql`: Define a estrutura do banco (DDL).
2. `Povoamento.Biblioteca.sql`: Insere dados de teste (DML).
3. `Consultas.Biblioteca.sql`: Contém consultas (Queries) para geração de relatórios.
